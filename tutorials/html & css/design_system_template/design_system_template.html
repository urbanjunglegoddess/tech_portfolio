Intro

Purpose

Our design system is a collection of reusable components, guided by clear standards, that can be assembled together to build any number of applications.

Scope

This design system is intended to be used by designers, developers, and other team members to create a cohesive user experience across all of our products.

Goals

  • Improve consistency across products
  • Speed up development time
  • Reduce design and technical debt

Audience

This design system is intended for use by designers, developers, and other team members who are responsible for creating and maintaining our products.
